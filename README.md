# SCP-KΣ-0001 :: «kosstar the 1st»

Статический сайт-досье в стилистике **Фонда SCP** (хоррор): глитчи, дрожащий
текст, сокрытые данные и «прорыв» аномалии из VRChat в реальность.

Сайт полностью автономный — **чистые HTML/CSS/JS без сборки**. Его можно сразу
развернуть на GitHub Pages.

## Файлы

```
index.html    # разметка досье
styles.css    # все стили и хоррор-эффекты
script.js     # boot-экран, глитчи, скролл-ревилы, аккордеоны
.nojekyll     # отключает обработку Jekyll на GitHub Pages
404.html      # страница ошибки в той же стилистике
```

## Как развернуть на GitHub Pages

1. Создайте репозиторий и загрузите в него эти файлы (корень репозитория).
2. Откройте **Settings → Pages**.
3. В разделе **Build and deployment** выберите:
   - **Source:** `Deploy from a branch`
   - **Branch:** `main` (или `master`) и папку `/ (root)`.
4. Сохраните. Через минуту сайт будет доступен по адресу
   `https://<ваш-логин>.github.io/<имя-репозитория>/`.

Файл `.nojekyll` уже включён, поэтому GitHub Pages отдаст файлы как есть.

## Локальный просмотр

Достаточно открыть `index.html` в браузере. Либо запустить простой сервер:

```bash
python3 -m http.server
# затем открыть http://localhost:8000
```
